
# Desktop School Application


With this project, students, teachers and exam grades at school can be followed.
To put it briefly, we can say that the School Information System.

## Used Technologies

 * Frontend
    - Windows Forms
 * Backend 
     - C#
 * Database
    - MS SQL
 * ORM
   - Entity Framework
